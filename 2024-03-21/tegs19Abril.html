Tags de código

Apresentar blocos de código em documentos HTML tornou-se uma atividade bastante comum, visto que existem inúmeros sites que precisam exigir trechos de código. E não apenas código, também precisamos mostrar a saída do código ao tentar replicar janelas de terminais

Blogs de tecnologia, plataformas de ensino de programação, documentações de software, provedores de API – há muitos contextos em que ser capaz de apresentar código ao usuário é um requisito vital.

code

code apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de código.

Assim "Lorem ipsum dolor sit amet consectetur adipisicing elit. Natus illo reiciendis placeat laboriosam atque temporibus commodi neque eum vero similique, omnis quibusdam esse error quibusdam"

samp

É usada para incluir texto em linha que representa a saída de amostra (ou citada) de um programa de computador. Seu conteúdo é normalmente renderizado usando a fonte padrão do navegador

pre

É a tag utilizada para representar texto pré-formatado. Um texto dentro desse elemento é tipicamente exibido em uma fonte não proporcional da mesma maneira em que o texto original foi disposto no arquivo. Espaços em branco são mantidos no texto da mesma forma em que este foi digitado.

kbd

Destina-se a marcar textos com um "botão".

Ctrl + Shift + P: Abre a caixa de comando, onde é possível pesquisar por funcionalidades específicas.

Ctrl + Shift + D: Duplica a linha atual.

Ctrl + Z: Desfaz a última ação.

Ctrl + Shift + X: Abre a barra de extensões, onde é possível instalar e gerenciar extensões do VS Code.

Ctrl + Shift + N: Abre uma nova janela do VS Code.
